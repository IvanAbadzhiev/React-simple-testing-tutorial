# How to run

Clone the repo
```
git clone 
```

Go to the folder with the project
```
cd weather-app
```

Install the dependencies of the project

```
npm install
```
or 

```
yarn install
```

Start the project
```
npm start
```
or 

```
yarn start
```

Run the tests on the project

```
npm test
```
or

```
yarn test
```
